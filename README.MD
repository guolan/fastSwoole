 安装步骤：

`1、下载框架：git clone git@github.com:panco95/fastSwoole.git 项目名称`

`2、安装所需类库：composer update`

`3、安swoole扩展：pecl install swoole`

`4、开启Http服务：php start.php start`

`5、浏览器访问 127.0.0.8888`
